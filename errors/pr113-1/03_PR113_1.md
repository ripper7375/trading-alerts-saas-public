The build is failing due to a syntax error: the Tailwind class bg-muted-foreground/30 does not exist. This error appears in your app/globals.css at lines where you use @apply bg-muted-foreground/30, such as:

```css
::-webkit-scrollbar-thumb {
  @apply bg-muted-foreground/30 rounded-full;
}
.status-inactive {
  @apply w-2 h-2 rounded-full bg-muted-foreground/30;
}
```

**Solution:**
Tailwind does not support opacity modifiers (e.g., /30) on custom color variables like bg-muted-foreground. To fix this, you should manually use a CSS color function for the background, applying alpha for opacity. For example, replace:

```css
@apply bg-muted-foreground/30;
```

with

```css
background-color: rgb(var(--muted-foreground) / 0.3);
```

or the appropriate CSS function for your color format.

Update both usages in your globals.css ([see file, ref 713de158b6e15536f153dfcac3829259f42f26f1](https://github.com/ripper7375/trading-alerts-saas-public/blob/713de158b6e15536f153dfcac3829259f42f26f1/app/globals.css)), like so:

```css
::-webkit-scrollbar-thumb {
  background-color: rgb(var(--muted-foreground) / 0.3);
  border-radius: 9999px;
}
.status-inactive {
  width: 0.5rem;
  height: 0.5rem;
  border-radius: 9999px;
  background-color: rgb(var(--muted-foreground) / 0.3);
}
```

This will resolve the build error.
